About Chicken Health Expert System

The Chicken Health Expert System is a web-based application designed to assist users in diagnosing and understanding chicken diseases. Nous-Hermes model has been fine-tuned using chicken related data to create an LLM capable of understanding and answering chicken-related user queries. By leveraging Django for backend operations and integrating geolocation services, it allows users to submit queries about chicken health. The system provides disease frequency analytics displayed both in a grid format, offering valuable insights into the prevalence of various diseases across different locations.

Additionally, the application features a "Find Vet" function that utilizes web browser geolocation to help farmers locate veterinary offices near them. This tool ensures that farmers can easily access professional help when needed.

The application is built with a focus on user-friendliness and efficient data presentation, utilizing Tailwind CSS for styling to ensure a responsive and visually appealing interface.
